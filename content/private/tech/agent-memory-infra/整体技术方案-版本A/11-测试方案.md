## 1. 测试分层策略

### 1.1 测试金字塔

```
                    ┌─────────────────┐
                    │    E2E Tests    │  ← 少量，验证核心业务流程（端到端）
                    │    (Playwright  │    覆盖率目标：核心用户路径 100%
                    │    + httpx)     │    运行频率：每次发版前
                    └────────┬────────┘
               ┌─────────────┴──────────────┐
               │     Integration Tests      │  ← 中等数量，验证组件协作
               │   (pytest + real DBs or    │    覆盖率目标：Pipeline 链路 ≥ 80%
               │    Docker containers)      │    运行频率：每次 PR 合并
               └────────────┬───────────────┘
      ┌──────────────────────┴───────────────────────┐
      │                 Unit Tests                   │  ← 大量，验证单一函数/类
      │  (pytest + mock，无外部依赖，运行时间 < 30s)   │    覆盖率目标：核心模块 ≥ 85%
      │                                              │    运行频率：每次 commit
      └──────────────────────────────────────────────┘
```

### 1.2 各层划分原则

| 层级 | 测试对象 | 外部依赖 | 运行速度 | 反馈价值 |
|------|---------|---------|---------|---------|
| **单元测试** | 单个函数/类方法，业务逻辑 | Mock 全部替代 | < 30s | 快速定位逻辑错误 |
| **集成测试** | 多个组件协作、数据库读写 | 真实 DB（Docker）| 1–5 分钟 | 验证数据流正确性 |
| **E2E 测试** | 完整 API 链路，用户场景 | 完整环境 | 5–20 分钟 | 验证功能符合预期 |
| **性能测试** | 检索接口吞吐量、延迟 | 生产级数据量 | 30 分钟+ | 发现性能瓶颈 |

#          --users 100 --spawn-rate 10 --run-time 5m

# 目标 SLO：
#   - P95 检索延迟 < 200ms
#   - P99 检索延迟 < 500ms
#   - 错误率 < 0.1%
#   - 吞吐量 >= 500 QPS（100 并发用户）

import json
import random
import time
from typing import Any

from locust import HttpUser, between, events, task
from locust.clients import ResponseContextManager


# ─── 测试数据 ─────────────────────────────────────────────────────────────

SAMPLE_QUERIES = [
    "如何处理VIP用户的退货投诉？",
    "企业账户的发票申请流程是什么？",
    "商品质量问题如何走赔偿流程？",
    "退款到账需要多少个工作日？",
    "顺丰/京东快递丢件如何赔偿？",
    "7天无理由退货的具体条件是什么？",
    "如何查询退款进度？",
    "用户要求补偿优惠券的处理策略？",
    "代码审查的最佳实践有哪些？",
    "如何优化大型 Python 项目的启动速度？",
]

SAMPLE_AGENTS = [f"agent_perf_test_{i:03d}" for i in range(50)]
SAMPLE_SESSIONS = [f"sess_perf_{i:05d}" for i in range(200)]


def get_auth_header() -> dict[str, str]:
    """获取性能测试专用认证头"""
    return {"Authorization": "Bearer perf_test_jwt_token"}


# ─── 性能测试用户 ──────────────────────────────────────────────────────────

class MemoryAPIUser(HttpUser):
    """
    模拟真实用户请求分布：
    - 80% 记忆检索（读密集）
    - 15% Episode 写入
    - 5% 技能列表查询
    """

    wait_time = between(0.5, 2.0)  # 每个用户请求间隔 0.5-2.0 秒
    headers = get_auth_header()

    def on_start(self):
        """用户开始时分配 Agent 和 Session"""
        self.agent_id = random.choice(SAMPLE_AGENTS)
        self.session_id = random.choice(SAMPLE_SESSIONS)

    @task(8)  # 权重 8：80% 概率
    def query_memory(self):
        """记忆检索请求"""
        query = random.choice(SAMPLE_QUERIES)
        payload = {
            "query": query,
            "agent_id": self.agent_id,
            "session_id": self.session_id,
            "memory_layers": ["procedural", "semantic"],
            "retrieval_config": {
                "top_k": 5,
                "min_relevance": 0.6,
                "search_modes": ["local", "global"],
            },
        }
        with self.client.post(
            "/api/v1/memory/query",
            json=payload,
            headers=self.headers,
            name="/api/v1/memory/query",
            catch_response=True,
        ) as response:
            self._validate_response(response, expected_status=200)

    @task(1)  # 权重 1：10% 概率
    def ingest_episode(self):
        """Episode 写入请求"""
        payload = {
            "agent_id": self.agent_id,
            "session_id": self.session_id,
            "write_mode": "async",
            "episode": {
                "episode_type": "task_execution",
                "user_input": random.choice(SAMPLE_QUERIES),
                "reasoning": "1. 识别意图 2. 检索相关知识 3. 生成回复",
                "actions": [
                    {"tool": "search_memory", "input": "退货", "output": "找到相关政策"}
                ],
                "response": "您好，根据我们的政策...",
                "outcome": random.choice(["success", "success", "success", "partial"]),
                "user_feedback": random.choice(["positive", "positive", "neutral"]),
            },
        }
        with self.client.post(
            "/api/v1/memory/ingest",
            json=payload,
            headers=self.headers,
            name="/api/v1/memory/ingest",
            catch_response=True,
        ) as response:
            self._validate_response(response, expected_status=202)

    @task(1)  # 权重 1：10% 概率
    def list_skills(self):
        """技能列表查询"""
        domains = ["customer_service", "code_review", "data_analysis"]
        domain = random.choice(domains)
        with self.client.get(
            f"/api/v1/skills?domain={domain}&page=1&page_size=10",
            headers=self.headers,
            name="/api/v1/skills",
            catch_response=True,
        ) as response:
            self._validate_response(response, expected_status=200)

    def _validate_response(
        self, response: ResponseContextManager, expected_status: int
    ):
        if response.status_code != expected_status:
            response.failure(
                f"期望状态码 {expected_status}，实际 {response.status_code}: "
                f"{response.text[:200]}"
            )
        elif response.elapsed.total_seconds() > 2.0:
            response.failure(
                f"响应时间过长: {response.elapsed.total_seconds():.2f}s > 2.0s"
            )
        else:
            response.success()


# ─── 自定义统计指标 ────────────────────────────────────────────────────────

@events.request.add_listener
def on_request(
    request_type, name, response_time, response_length, exception, **kwargs
):
    """记录 P95/P99 延迟到自定义指标"""
    if exception:
        print(f"[PERF-FAIL] {name}: {exception}")


@events.test_stop.add_listener
def on_test_stop(environment, **kwargs):
    """测试结束后输出 SLO 评估报告"""
    print("\n===== SLO 评估报告 =====")
    for name, stats in environment.runner.stats.entries.items():
        p95 = stats.get_response_time_percentile(0.95)
        p99 = stats.get_response_time_percentile(0.99)
        error_rate = stats.fail_ratio

        p95_pass = p95 < 200 if "query" in str(name) else p95 < 500
        p99_pass = p99 < 500 if "query" in str(name) else p99 < 1000

        print(f"\n接口: {name}")
        print(f"  P95: {p95:.0f}ms {'[OK]' if p95_pass else '[FAIL]'}")
        print(f"  P99: {p99:.0f}ms {'[OK]' if p99_pass else '[FAIL]'}")
        print(f"  错误率: {error_rate:.3%} {'[OK]' if error_rate < 0.001 else '[FAIL]'}")
        print(f"  RPS: {stats.current_rps:.1f}")

### 10.1 各模块覆盖率目标

| 模块 | 文件路径 | 最低覆盖率 | 说明 |

| **Episode Memory** | `src/memory/episode_memory.py` | 85% | 写入管道核心 |
| **Procedural Memory** | `src/memory/procedural_memory.py` | 85% | Skill 读写逻辑 |
